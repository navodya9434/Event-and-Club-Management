# Event-and-Club-Management

## University Event and Club Management System

A full-stack web application developed to streamline the management of university clubs, events, and student participation. The system provides a centralized platform for students and organizers to create, manage, and engage with club activities and university events efficiently.

### Features

* User registration and authentication
* Club creation, meeting scheduling  and management
* Event creation, scheduling, and management
* Role-based access for organizers and students
* Responsive and user-friendly interface
* Database-driven event and membership management

### Technologies Used

* Backend: Spring Boot
* Frontend: React.js
* Database: MySQL
* Development Tools: MySQL Workbench, VS Code, Git

### Project Objectives

* Improve communication between students and club organizers
* Simplify event planning and participant management
* Provide a centralized system for managing university extracurricular activities

### Key Learning Outcomes

* Full-stack application development
* Frontend-backend communication
* Database design and management
* Component-based UI development with React
* Software engineering best practices
